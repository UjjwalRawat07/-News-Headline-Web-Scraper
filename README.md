
# News Headline Web Scraper

# Objective
This project is a simple web scraper built using Python that extracts top headlines from a news website and saves them into a text file.

# Tools Used
- Python 3.14.3
- requests
- BeautifulSoup

# Key Concepts
- HTTP GET Requests
- Web Scraping
- HTML Parsing
- File Handling

# How It Works
1. Sends a GET request to a news website.
2. Parses the HTML using BeautifulSoup.
3. Extracts headlines from tags.
4. Saves them into a file named `headlines.txt`.
